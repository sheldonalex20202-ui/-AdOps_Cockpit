# Glossary

## Ad account

Рекламный кабинет Meta/Facebook.

## Pool

Группа рекламных кабинетов под тест, scale, проблемные состояния или другую рабочую категорию.

## Health check

Проверка готовности кабинета по статусу, токену, биллингу, лимиту и свежести синка.

## Launch job

Задача массового создания рекламной структуры на выбранных кабинетах.

## Creative

Креатив, который участвует в запуске. Может иметь `geo` и `zGroup`.

## Z group

Группа креативов по Z-номеру, например `Z1`, `Z2`, `Z3`.

## Headline set

Набор заголовков, сопоставленных Z-группам.

## Desktop callback

Локальный HTTP endpoint, который desktop запускает временно для получения JWT после web-авторизации.

## Entitlements

Набор разрешений и лимитов, вытекающих из плана подписки.
