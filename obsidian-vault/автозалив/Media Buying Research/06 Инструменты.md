## Трекеры

### Keitaro — keitaro.io

**Роль:** TDS (Traffic Distribution System) + трекер + клоака

**Функции:**
- 120+ готовых шаблонов источников трафика (включая Facebook)
- Flows (потоки): распределение трафика по условиям (источник, устройство, ГЕО, поведение)
- Клоаки: фильтр по User-Agent, IP, referrer → белая страница модераторам / чёрная пользователям
- S2S постбэки: передача конверсий от CPA-сети → Keitaro → Facebook CAPI
- Статистика: клики, лиды, CR, ROI в разрезе по любому параметру

**Интеграция с Facebook:**
1. **Pull расходов**: Settings → Integrations → Add Facebook → Account ID + API token
2. **Push конверсий (CAPI)**: Pixel ID + CAPI Access Token из Events Manager

**Важно:** `fbclid` НЕ добавлять вручную как макрос — формируется Facebook автоматически; ручное добавление ломает запись.

### Binom

Аналог Keitaro, популярный в западном арбитраже. Self-hosted, нет SaaS-версии.

### Redtrack / Voluum

Облачные трекеры, популярны в западном e-commerce и нутре. Более дорогие.
