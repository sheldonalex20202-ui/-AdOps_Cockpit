# Media Buying Research — Обзор

Этот раздел базы знаний документирует реальные практики медиабаеров: как устроен залив, какие инфраструктуры используются, как разные команды называют кампании, какие инструменты применяют. Цель — сделать AdOps Cockpit максимально адаптивным под разные команды и стили работы.

## Что такое залив

**Залив** — процесс запуска и ведения платных рекламных кампаний в Facebook/Meta Ads с одного или множества рекламных аккаунтов с целью генерировать трафик на оффер (партнёрский продукт).

**Автозалив** — программный инструмент (Dolphin Cloud, FBTool, Nooklz), позволяющий:
- Настраивать и запускать кампании одновременно на множестве аккаунтов из единого интерфейса
- Дублировать кампании в один клик и применять пресеты (шаблоны / связки)
- Мониторить статус аккаунтов, биллингов, статистики
- Устанавливать авто-правила (автопауза, автомасштаб)
- Массово привязывать платёжные инструменты

## Ключевые термины

| Термин | Значение |
|---|---|
| **Связка** | Комбинация: оффер + ГЕО + крео + лендинг + аккаунт |
| **Оффер** | Партнёрский продукт из CPA-сети |
| **Крео** | Рекламный материал (видео/изображение/карусель) |
| **Прелендинг** | Промежуточная страница перед лендингом оффера |
| **ГЕО** | Целевая страна |
| **Заливала** | Исполнитель: запускает кампании по инструкциям байера |
| **Байер** | Медиабайер: самостоятельно выбирает и оптимизирует связки |
| **Фармер** | Создаёт и прогревает аккаунты |
| **Клоака** | Система: модераторам — белая страница, пользователям — реальный оффер |
| **BM** | Business Manager — инструмент централизованного управления |
| **Кинг / Мамка** | Высокодоверенный аккаунт, распределяющий траст на рабочие аккаунты |
| **Расходник** | Аккаунт, который предполагается потерять (забанят) |
| **Z-группа** | Батч/волна креативов одного поколения теста (Z1, Z2, ...) |
| **Нейминг** | Система именования кампаний/адсетов/объявлений |
| **ABO** | Ad Set Budget Optimization — бюджет на уровне адсета |
| **CBO** | Campaign Budget Optimization — бюджет на уровне кампании |
| **CAPI** | Conversions API — серверная передача конверсий в Facebook |
