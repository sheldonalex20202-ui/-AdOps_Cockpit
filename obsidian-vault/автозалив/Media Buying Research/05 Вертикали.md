# Вертикали

## Нутра (Nutra)

### Характеристика
- Продукты для здоровья: похудение, суставы, мужское здоровье, кожа, диабет
- Тип конверсии: COD (Cash on Delivery) — оплата при получении, или SS (Straight Sale)
- Выплата: $15–35 за подтверждённый заказ
- Уровень сложности: средний (меньше клоаки чем в гемблинге)

### Популярные ГЕО
- Tier-1: DE, FR, IT, ES, PL (Европа)
- Tier-2: BR, MX, PH, IN
- CIS: RU, UA, KZ

### Воронка
```
Крео (боль/трансформация) → Прелендинг (статья, "новостной" стиль)
  → Лендинг (форма заказа) → Колл-центр → COD-подтверждение
```

### Нейминг
```
Кампания:  NUTRA_DE_SlimPro_BeforeAfter_FB_251015
Адсет:     DE_F35-55_INT_WeightLoss_Feed
Объявление: VID_BA_DoctorAngle_V1
```

### Ключевые углы (angles)
- `BA` — До/После (Before/After) — самый конвертящий
- `DR` — Доктор/Медицинский авторитет
- `TEST` — Отзыв реального человека
- `NEWS` — Новостной нативный стиль
- `QUIZ` — Квиз / симптом-чеклист
- `PAIN` — Боль + решение

### Подкатегории и коды
| Код | Подкатегория |
|---|---|
| `WL` | Weight Loss (похудение) |
| `JOINT` | Суставы |
| `MHEALTH` | Мужское здоровье |
| `SKIN` | Кожа |
| `DIAB` | Диабет |
| `VISION` | Зрение |
| `HAIR` | Волосы |

### Особенности запуска
- Можно лить без клоаки на "белые" нутра-продукты
- Серая нутра (чудо-таблетки, фарма) — требует клоаку
- Прелендинг критичен: повышает Click2Reg на 30–60%
- Лучшие плейсменты: Facebook Feed, Instagram Feed, Reels

## Гемблинг (Gambling)

### Характеристика
- Онлайн-казино, покер, слоты
- Тип конверсии: FTD (First Time Deposit)
- Выплата: $30–200+ за первый депозит
- Уровень сложности: высокий (клоака обязательна)

### Популярные ГЕО
- Tier-1: DE, AT, CH, AU, CA
- Tier-2: PL, CZ, RO, HU
- LatAm: BR, CL, AR, MX

### Воронка
```
Крео (сцена выигрыша / бонус) → [Клоака]
  → Прелендинг (опциональный) → Страница казино → Регистрация → Депозит
```

### Запуск на приложения (app-based)
Самая популярная схема в 2024–2026:
1. Объявление ведёт на страницу установки приложения (Play Market / App Store)
2. Пользователь устанавливает приложение казино
3. Deeplink ведёт сразу к форме регистрации
4. Конверсия = FTD

**Цель кампании:** `OUTCOME_APP_PROMOTION`

### Нейминг
```
Кампания:  GMB_PL_BetAndreas_ABO_20260101
Адсет:     PL_M18-35_Broad_Adv+
Объявление: VID_WinStory_V1
```

### Нейминг для deeplink
```
fb_DE_betandreas_cr001_251015
[источник]_[ГЕО]_[оффер]_[крео-ID]_[дата]
```

### Ключевые углы
- `WIN` — Сцена выигрыша (обычный человек выигрывает крупную сумму)
- `BONUS` — Приветственный бонус, фриспины
- `FOMO` — Срочность: "только сегодня", "осталось 5 мест"
- `GIRL` — Привлекательная девушка + казино (спорный, работает на мужчин)
- `SLOT` — Видео вращения слота с крупным выигрышем

### Особенности запуска
- Клоака **обязательна** — FB не одобряет гемблинг в большинстве ГЕО
- Лицензированное гемблинг приложение может запускаться без клоаки (особый доступ)
- Аккаунты под гемблинг — расходники; иметь запас прогретых аккаунтов
- Лучшие аудитории: BROAD / Advantage+ (FB сам находит игроков)

## Крипто (Crypto)

### Характеристика
- Торговые платформы, инвестиционные сервисы, Bitcoin-новости
- Тип конверсии: FTD или Lead (форма для звонка колл-центра)
- Выплата: $400–1200+ за депозит
- Уровень сложности: очень высокий (строгая клоака, риск бана страницы)

### Воронка
```
Крео (новостной стиль / знаменитость) → Прелендинг (news-статья с "сенсацией")
  → Лендинг (форма регистрации) → Колл-центр → Депозит
```

### Нейминг
```
Кампания:  CRYPT_PL_BitcoinTrader_NewsAngle_FB_251015
Адсет:     PL_M35-65_INT_Finance-Crypto_Feed
Объявление: VID_CelebEndorse_ElonMusk_V2
```

### Ключевые углы
- `NEWS` — Новостной нативный ("Bitcoin достиг $X — эксперты говорят...")
- `CELEB` — Знаменитость инвестирует в платформу
- `FOMO` — "Последний шанс войти в рынок"
- `AUTOPILOT` — "Торговый бот зарабатывает за вас"
- `AVER` — "Средний человек заработал X за месяц"

### Особенности запуска
- Самые высокие выплаты, но самые жёсткие требования к клоаке
- Целевая аудитория: мужчины 35–65, интересы к финансам/инвестициям
- Pre-lender (новостная статья) критически важен для конверсии

## Дейтинг (Dating)

### Характеристика
- Сайты знакомств, приложения
- Тип конверсии: SOI (Single Opt-In) / DOI (Double Opt-In) / PPS (Pay Per Sale)
- Выплата: $1–30 за регистрацию (зависит от типа)
- Уровень сложности: низкий–средний

### Воронка
```
Крео (привлекательный человек / "кто-то написал тебе") → Лендинг регистрации → Конверсия
```

### Нейминг
```
Кампания:  DAT_US_OneLove_GirlMsg_FB_251115
Адсет:     US_M25-45_INT_Dating_AllPlacements
Объявление: IMG_GirlMessage_V1
```

### Ключевые углы
- `MSG` — "Кто-то написал тебе сообщение" (скриншот или уведомление)
- `PHOTO` — Фото привлекательного человека с текстом
- `QUIZ` — "Ты ей нравишься?" / "Найди пару"
- `MATCH` — "5 человек в твоём городе ждут тебя"

### Особенности запуска
- SOI: самая низкая выплата, самый высокий объём
- DOI: нужно подтверждение email, выплата выше
- Adult-дейтинг — клоака обязательна
- Лучшие аудитории: мужчины 25–55, интересы к отношениям

## Товарка (E-commerce)

### Характеристика
- Физические товары: одежда, аксессуары, товары для дома
- Тип конверсии: покупка (Purchase)
- Выплата: RevShare (% от чека) или фиксированная CPA
- Уровень сложности: низкий (белая тематика)

### Воронка
```
Крео (товар / демонстрация) → Лендинг-магазин → Корзина → Оплата
```

### Нейминг
```
Кампания:  ECO_US_SummerShoes_CBO_20260601
Адсет:     US_F25-45_INT_Fashion_BROAD
Объявление: VID_ProductDemo_V1
```

### Ключевые углы
- `DEMO` — Демонстрация продукта
- `UNBOX` — Анбоксинг
- `USE` — Как использовать продукт
- `LIFESTYLE` — Стиль жизни с продуктом
- `SOCIAL` — Отзывы и социальное доказательство

## White Hat vs Grey Hat: различия в организации

### White Hat
- Полная прозрачность в нейминге: бренд, продукт, цель, аудитория
- Без клоаки, без fake-news углов
- Нейминг в Ads Manager = реальный нейминг в аналитике
- Пример: `ACME_CONV_TOF_HealthDrink_CBO_US_Jan26`

### Grey Hat (нутра, гемблинг, крипто)
- **Двухслойная система**: нейтральный/обфусцированный нейминг в Ads Manager + полная аналитика в трекере через UTM
- Названия кампаний в Ads Manager часто нейтральные, чтобы не триггерить автомодерацию
- Реальные метаданные: оффер, ГЕО, угол — хранятся во внутренней таблице
- Крео-файлы в кабинете называются обобщённо (`video_01.mp4`), но внутренний ID маппится на полное имя
- Пример в Ads Manager: `Campaign_20260115_v1`
- В трекере/таблице: `GMB_PL_BetAndreas_WinStory_Z2_260115`
